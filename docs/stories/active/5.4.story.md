## Story
**As a** dono,
**I want** ver a media de preco de combustivel por cidade e estado calculada a partir dos lancamentos reais dos motoristas,
**so that** eu possa comparar se os abastecimentos recentes estao dentro ou acima da media historica da regiao e tomar decisoes sobre rotas e postos.

## Acceptance Criteria

1. Existe uma pagina ou secao acessivel pelo dono em `/dashboard/configuracoes/combustivel` (ou equivalente ja existente) que exibe a media de preco por regiao.
2. A exibicao usa os dados da view `vw_media_combustivel_regiao` e agrupa por `estado` e `cidade`.
3. Para cada regiao com dados, exibe: estado/UF, cidade, tipo de combustivel, preco medio (R$/L), preco minimo, preco maximo, quantidade de abastecimentos e data do abastecimento mais recente.
4. Os dados sao filtrados por `empresa_id` via RLS (automatico pelo Supabase client).
5. Quando uma regiao possui menos de 3 abastecimentos, exibe indicador visual de "dados insuficientes" (ex: badge amarelo ou asterisco com nota) para sinalizar que a media pode nao ser representativa.
6. A pagina/secao exibe estado vazio quando nao ha nenhum abastecimento com cidade/estado preenchidos, com mensagem orientativa.
7. Os dados sao atualizados automaticamente (sem cache manual): cada acesso a pagina busca dados atuais da view.
8. Acesso restrito a `role = dono` ou `role = admin`. Motorista nao acessa esta pagina.
9. A view `vw_media_combustivel_regiao` criada na Story 5.1 e usada diretamente (sem RPC adicional).
10. Layout responsivo e legivel em mobile (375px), com dados tabulares em cards ou layout adaptado para tela pequena (sem tabela horizontal com scroll).

### View disponivel
A view `vw_media_combustivel_regiao` foi criada na Story 5.1 com o seguinte schema:
```
vw_media_combustivel_regiao
  empresa_id         UUID
  estado             CHAR(2)
  cidade             TEXT
  tipo_combustivel   combustivel_tipo
  total_abastecimentos  BIGINT
  preco_medio_litro  NUMERIC(10,4)   -- R$/litro, ex: 6.4590
  preco_min_litro    NUMERIC(10,4)
  preco_max_litro    NUMERIC(10,4)
  total_litros       NUMERIC
  total_centavos     BIGINT
  primeira_data      DATE
  ultima_data        DATE
```

### Query da server action
```typescript
// app/(dashboard)/configuracoes/combustivel/actions.ts
export async function getMediaPorRegiao() {
  const supabase = createServerSupabaseClient()

  // Verificar role antes de consultar
  const { data: profile } = await supabase
    .from('usuario')  // ou tabela de perfil existente
    .select('role')
    .single()

  if (!['dono', 'admin'].includes(profile?.role)) {
    return { error: 'Acesso nao autorizado' }
  }

  const { data, error } = await supabase
    .from('vw_media_combustivel_regiao')
    .select('*')
    .order('estado', { ascending: true })
    .order('cidade', { ascending: true })

  return { data, error }
}
```

- Ex: R$ 6,46/L
