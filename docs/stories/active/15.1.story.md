## Story
**Como** motorista de 55 anos usando o FrotaViva,
**Quero** que todas as palavras na tela tenham acentuacao correta em portugues,
**Para que** eu nao sinta que o sistema e malfeito ou "gringo" quando leio "Situacao" sem til ou "Saida" sem acento.

## Acceptance Criteria

1. As 84 strings sem acentuacao identificadas na auditoria UX (#5) sao corrigidas. Inclui pelo menos: "Situacao" -> "Situacao", "Saida" -> "Saida", "Distancia" -> "Distancia", "Opcoes" -> "Opcoes" e demais do dicionario.
2. A label "observacao " em ViagemForm tem o whitespace extra removido (AC da auditoria @ux #19).
3. Um script `npm run check-ptbr` existe e executa uma busca regex por um dicionario de 30-40 palavras criticas sem acento (ex: `Saida`, `Situacao`, `Opcao`, `Distancia`, `Veiculo`, `Numero`, etc) em todos os arquivos `.tsx`, `.ts` e `.json` em `components/` e `app/`.
4. O script retorna exit code 0 se nenhuma string sem acento for encontrada e exit code 1 com listagem de ocorrencias se houver.
5. O script roda no hook `pre-commit` (Husky, da story 14.1). Um commit com "Saida" em um label sera bloqueado.
6. Strings em variaveis JavaScript, nomes de funcoes, ou IDs nao sao afetadas pela auditoria (somente textos visiveis ao usuario final em JSX).
7. `npm run typecheck && npm run lint && npm run build` passam.
8. Uma revisao visual de 5 paginas do dashboard (home, viagens, fechamentos, motoristas, bi) confirma que nao sobrou nenhum texto sem acento visivel.

## Dev Notes

- O dicionario de palavras criticas deve conter variantes de case: `Saida`, `saida`, `SAIDA`. A regex deve ser case-insensitive.
- Cuidado com falsos positivos: nome de funcao `getSaida()` nao deve ser apontado. O script deve filtrar apenas conteudos de JSX string literals e atributos de template literal renderizados.
- Se o filtro for muito complexo, uma alternativa aceitavel e: o script alerta, mas o @dev avalia manualmente se e texto visivel ou codigo interno. Script com exit code 1 lista candidatos, desenvolvedor confirma.
- Publico 55+: acentuacao e questao de confianca. Sem acento = "os meninos nao sabem portugues". E percebido como descuido grave.
- Esta story nao tem dependencia de nenhuma outra. Pode ser a primeira a rodar no Sprint 2.
- Arquivos previsivelmente afetados: 30+ arquivos em `components/` e `app/`, `scripts/check-ptbr.js` (novo), `package.json`, `.husky/pre-commit`.
