## Dev Notes

### Modelo escolhido: gemini-2.5-flash
A doc oficial confirmou que `gemini-2.5-flash` tem o free tier mais generoso no Google AI Studio e suporta tool use + streaming nativamente via `@ai-sdk/google`. Import direto: `import { google } from '@ai-sdk/google'`. Nao precisa de `createGoogleGenerativeAI` — o factory default le `GOOGLE_GENERATIVE_AI_API_KEY` do env.

### Decisao Upstash confirmada
Ver `docs/stories/frotaviva-copilot/rate-limit-decision.md`. Two sliding windows: `10/60 s` (por minuto) + `200/1 d` (por dia), consultados em paralelo via `Promise.all`. O mais restritivo decide. Fail-open em caso de erro do Upstash (quota/network) — logga `console.warn` e deixa passar para nao bloquear usuario legitimo. Story 9.5 vai trocar esse warn pelo `logger.warn` dedicado.

### Bloqueios manuais pro stakeholder (AC-6)
AC-5 esta OK (`.env.example` atualizado com placeholders). **AC-6 exige acao manual:**
1. Criar API key em https://aistudio.google.com/apikey e copiar para `GOOGLE_GENERATIVE_AI_API_KEY` em `.env.local`
2. Criar Redis database em https://console.upstash.com (free tier) e copiar `UPSTASH_REDIS_REST_URL` + `UPSTASH_REDIS_REST_TOKEN` para `.env.local`

Ate isso ser feito, a story 9.5 (primeira chamada real) nao roda. 9.2/9.3/9.4 independem dessas keys (scaffold + tools + testes mockados).
