## Story
**As a** dono/admin do FrotaViva,
**I want** um modal que exibe as credenciais geradas do motorista com opcoes de copiar e enviar pelo WhatsApp,
**so that** eu possa repassar o acesso ao motorista de forma rapida e segura logo apos o cadastro.

## Acceptance Criteria

1. Existe o componente `CredenciaisModal` em `components/motoristas/CredenciaisModal.tsx`.
2. O modal recebe as props: `credenciais: { email: string; senha: string }` e `onClose: () => void`.
3. O modal exibe um titulo "Motorista cadastrado com sucesso!".
4. O modal exibe email e senha em um card destacado com fundo neutro (cinza claro ou equivalente no tema do projeto), em fonte monoespaco para facilitar a leitura.
5. Existe um botao "Copiar Credenciais" que ao ser clicado copia o texto `Email: {email}\nSenha: {senha}` para a area de transferencia usando a Clipboard API (`navigator.clipboard.writeText`).
6. Apos copiar, o botao exibe "Copiado!" por 2 segundos e volta ao texto original.
7. Existe um botao "Enviar por WhatsApp" que abre `https://wa.me/?text=` com a mensagem pre-formatada: "Ola! Suas credenciais de acesso ao FrotaViva:\nEmail: {email}\nSenha: {senha}\n\nAcesse: {URL do app}". O link deve abrir em nova aba (`target="_blank"`).
8. Existe um aviso em destaque: "Atencao: Esta senha sera exibida apenas uma vez. Anote-a agora ou envie ao motorista antes de fechar."
9. Existe um botao "Ir para Lista de Motoristas" que chama `onClose()` e redireciona para `/motoristas` via `router.push`.
10. O modal e exibido como overlay sobre toda a pagina (backdrop semi-transparente), centralizado verticalmente e horizontalmente.
11. Clicar fora do modal NAO fecha o modal (previne fechamento acidental antes de anotar a senha).
12. O modal e responsivo e funciona em viewport de 375px (mobile-first).
13. O botao "Ir para Lista de Motoristas" e o unico medio de fechar o modal.

## Dev Notes

### URL do App na Mensagem WhatsApp
Usar `window.location.origin` para montar a URL base, ou usar variavel de ambiente `NEXT_PUBLIC_APP_URL` se existir no projeto. Verificar `.env.example` para checar se essa variavel existe.

### Clipboard API em HTTPS
A Clipboard API (`navigator.clipboard.writeText`) requer contexto seguro (HTTPS ou localhost). Em producao funciona normalmente. Em desenvolvimento com `http://localhost` tambem funciona. Nao ha problema adicional a tratar.

### Mensagem WhatsApp Encodada
A mensagem para o deep link deve ser encodada com `encodeURIComponent`:
```typescript
const mensagem = encodeURIComponent(
  `Ola! Suas credenciais de acesso ao FrotaViva:\nEmail: ${email}\nSenha: ${senha}\n\nAcesse: ${window.location.origin}`
);
const url = `https://wa.me/?text=${mensagem}`;
```

### Backdrop sem fechamento ao clicar fora
```tsx
<div className="fixed inset-0 bg-black/50 z-50 flex items-center justify-center">
  <div className="bg-white rounded-lg p-6 max-w-md w-full mx-4" onClick={e => e.stopPropagation()}>
    {/* conteudo */}
  </div>
</div>
```
O click no backdrop externo nao deve propagar para o `onClose`. Nao registrar nenhum handler de click no backdrop.

### Fonte Monoespaco
Usar `font-mono` do Tailwind para o card de credenciais. Facilita a leitura e copia manual das credenciais.

### Dependencias
Esta story depende da Story 8.2 que instancia o modal. O componente pode ser desenvolvido independentemente e integrado na Story 8.2.
