**Test Failure Detail:**
- File: `__tests__/lib/utils/validate-placa.test.ts` line 83
- Test: `maskPlaca > truncates at 7 chars`
- Expected: `ABC-1234`, Received: `ABC1234`
- Root Cause: `maskPlaca('ABC12345678')` truncates to 7 chars `ABC1234` which is ambiguous (could be antigo without the 4th char being tested for antigo detection). The `maskPlaca` function at line 49 returns `raw.slice(0, 7)` when `raw.length > 7`, but does not re-apply the dash logic on the truncated result.
- **Severity:** LOW -- this is a display mask edge case on invalid input (8+ chars), not a data integrity issue. The `normalizePlaca()` and `validatePlaca()` functions work correctly.
- **Note:** This test failure is in shared utility code (Story 1.4 depends on it too).
